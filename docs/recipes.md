# Task Recipes

Recipes are short workflow cards for common task types.

Generate starter recipes with:

```bash
codex-harness project init
```

The first recipe set covers:

```text
bugfix
feature
refactor
take-home
open-source-pr
```

Recipes do not replace Codex planning. They give Codex a starting rhythm for
common work so that each new project does not need to reinvent the same process.
